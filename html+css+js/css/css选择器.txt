1.文档使用文档内嵌样式表和外部样式表，会使用选择器，例如：
	<style type="text/css">
				a { font-size:40px; color:#FFFFFF; }
				//这个就是选择器
	</style>
2.css基本选择器：6种
	选择所有元素：在大括号前面使用 * 号代表所有，即对所有标签进行样式修改
	根据类型选择元素：即根据在大括号前面使用的标签类型选择对应标签来进行样式修改
	根据类选择元素：当标签使用了class属性后，可在大括号前面使用 . +类的值来对对应标签进行样式修改
	根据id选择元素：当标签使用了id属性后，可在大括号前面使用 # +id的值来对对应标签进行样式修改
	根据属性选择元素：可以通过标签中的某个属性，在大括号前面使用[ 属性名称 ]或者[ 属性名称 = "属性值"]来对对应标签进行样式修改
	其他选择器：

3. : 选择器 即冒号选择器
	通过在原有选择器的基础，在新增一个 原有选择器大括号前面的字符 加上 :  触发事件名称 {}来实现改变原来样式状态
	例如：
		a { font-size:40px; color:#FFFFFF; }
		a : hover { font-size:80px; color:#111111; }
		实现当鼠标移动到a标签时，样式发生改变

	冒号后面的触发事件名称：
		hover ： 当鼠标移动到标签时，样式改变
