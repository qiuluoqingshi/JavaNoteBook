1.<table>标签：创建表格
	属性：
		border：边框属性，例如border="1px" 代表边框大小为1像素
		align：水平对齐方式，例如align="center" 代表水平居中 
2.<tr>标签：表格的行

3.<th>标签：一行中的一个单元格，一般是用在表格第一行，即属性名称，加粗居中处理字体
   <td>标签：一行中的一个单元格，一般是用在除表格第一行外，即属性，居左处理字体
	属性：
		colspan：合并行单元格，例如colspan = "2" 合并两个行单元格
		rowspan：合并列单元格，例如rowspan = "2"合并两个列单元格

4.<thead>标签：代表表头，和<th>搭配使用
   <tbody>标签：代表表格主体，和<td>搭配使用
   <tfoot>标签：代表表脚
